# AspNetCoreDashboard.Generators

为 `IUiModule` 实现类生成 **内容命名空间与路径前缀常量** 的源生成器，减少手写字符串与 `[UiModule]` 不一致的风险。

| 属性 | 值 |
|------|-----|
| NuGet | `AspNetCoreDashboard.Generators` |
| TFM | `netstandard2.0` |
| 类型 | 开发依赖（`DevelopmentDependency`） |

## 功能概览

对实现 `IUiModule` 的类，自动生成 `partial` 分部类，包含：

```csharp
private const string GeneratedContentNamespace = "MyCompany.Dashboard.Content";
private const string GeneratedPathPrefix = "/Dashboard";
```

生成逻辑：

- 读取 `[UiModule("/Path", ContentNamespace = "...")]` 特性值（若存在）
- 无特性时：`PathPrefix` 由类名推导（去掉 `UiModule` 后缀，加前导 `/`）
- 无特性时：`ContentNamespace` 默认为 `{ClassName}.Content`

## 使用方法

### 1. 引用包

```xml
<PackageReference Include="AspNetCoreDashboard.Generators" Version="3.6.0">
  <PrivateAssets>all</PrivateAssets>
  <IncludeAssets>runtime; build; native; contentfiles; analyzers; buildtransitive</IncludeAssets>
</PackageReference>
```

### 2. 将模块类声明为 partial

```csharp
[UiModule("/Dashboard", ContentNamespace = "MyCompany.Dashboard.Content")]
public sealed partial class DashboardUiModule : IUiModule
{
    public string PathPrefix => GeneratedPathPrefix;

    public void Configure(IUiModuleRegistration builder)
    {
        builder.MapEmbeddedUi(typeof(DashboardUiModule).Assembly, GeneratedContentNamespace)
               .MapFallbackToIndex($"{GeneratedContentNamespace}.index.html");
    }
}
```

生成文件位于编译输出：`{ClassName}.ContentNamespace.g.cs`（`// <auto-generated />`）。

## 与 Analyzers 的配合

- 生成器提供约定常量；**ACD006** 分析器可检测 `PathPrefix` 属性与 `[UiModule]` 是否一致。
- 建议与 `AspNetCoreDashboard.Analyzers` 一并引用。

## 注意事项

- 类必须实现 `AspNetCoreDashboard.Abstractions.IUiModule`，否则不会生成代码。
- 生成器**不**嵌入文件、不注册路由；仅生成常量。
- 修改 `[UiModule]` 或类名后需重新编译以更新生成代码。
- 可选包：不引用时可继续手写字符串，无运行时影响。
